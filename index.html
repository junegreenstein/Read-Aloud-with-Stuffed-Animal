Project description

The app allows a user to create a story, and then a virtual stuffed animal will read the story aloud. The user enters a title, author, and story content over several pages. Then, the user is prompted to draw a picture for the cover of the book. The story is saved, so the user can come back to previously written stories between sessions of the app. The app is targeted toward children.
